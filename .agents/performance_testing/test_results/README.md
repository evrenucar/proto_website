# Performance Test Results

Generated Cosmoboard benchmark runs are written here under timestamped folders:

```text
.agents/performance_testing/test_results/<timestamp>/
```

Each run folder should contain `results.json`, `summary.csv`, `report.md`, and, when requested with `--trace`, `trace.zip`.

This directory ignores generated run folders by default. Add a specific run intentionally when it is needed for review, regression comparison, or an audit record.
