# completed_research/

Research artifacts that have been consumed by a decision in `holistic_planning/` or by a feature spec in `feature_implementation/`. Files are not edited after archiving beyond a single `Outcome:` line at the top of the file noting where the decision landed.

To find an archived artifact, check `../research_areas.md` for the link, or search this folder by topic. The original document template (abstract, main findings, visuals, body, references) is preserved as the snapshot of what was known at the time the decision was made.

See [`../README.md`](../README.md) for the document template and lifecycle.
